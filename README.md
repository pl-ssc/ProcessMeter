# ProcessMeter

Одностраничное приложение для заполнения трудоёмкости операций в бизнес‑процессах.

## Что внутри
- Frontend: React + Glide Data Grid (табличный инлайн‑ввод)
- Backend: Fastify + Postgres
- БД: Postgres

## Быстрый старт (локально)
1. Поднять Postgres (Docker) и выполнить миграцию
2. Создать пользователя
3. Запустить фронт и бэк

См. подробности в `docs/SETUP.md`.
